# BOUNCE

## Instruktioner
BOUNCE!
- Styr den studsande vita bollen
- Undvik de röda fallande objekten
- Överlev så länge som möjligt!
- *Easter egg vid 2000 poäng*

## Kontroller
- Styr spelaren med pilarna
- Starta spelet med S
- Stäng av spelet med Esc
- Starta om spelet med R
- Pausa spelet med P
- Se tutorial med T
